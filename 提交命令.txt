初始化git
1.设置用户名
git config --global user.name'wandun'

2.设置用户名邮箱
git config --global user.email '1716221479@qq.com'

3.查看设置
git config -list




-- 查看工作区
git status   

-- 工作区到暂存区
git add hello.php
git add sorm.java

-- 暂存区到Git仓库中
git commit -m "提交的描述"

-- 提交到远程仓库
git remote add origin 

-- 将数据提交到远程Git仓库中
git push

-- 将远程的代码下载到本地 
git clone

-- 删除文件，删除文件要分2部分
1.删除本地文件  rm -rf 文件名
2.删除远程Git官网文件 git rm 文件名
3.最后也必须提交，否则没有效果 git commit -m "相关描述"
4.如果你希望同步到远程Git服务端使用git push 即可
